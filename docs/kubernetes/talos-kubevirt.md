Running KubeVirt on a single-node Talos Linux cluster.

## Overview

Talos Linux is a minimal, immutable OS designed for Kubernetes. This documents setting up a single-node cluster to run KubeVirt for VM workloads.

## Prerequisites

- Hardware with virtualization support (Intel VT-x / AMD-V)
- Talos Linux ISO
